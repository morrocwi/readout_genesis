The full original `phi_schedule_pagecurve_v5_3.py` was visible through the user's File Library as indexed content, but the connector result was not a local byte-addressable file and long-content retrieval was truncated. It is therefore **not represented as an exact original source file** in this ZIP. The exact recovered CSV/JSON evidence, independent audit script, and figure outputs are included so that every numerical claim made in the standalone manuscript can still be checked.
