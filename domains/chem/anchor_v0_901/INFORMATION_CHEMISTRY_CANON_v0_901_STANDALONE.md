# Information Chemistry Canon v0.901

## Standalone Root Proof Layer

This release contains everything required to inspect and rerun its claims without network access:

- a snapshot of the root source;
- the clean v0.1 anchor;
- the frozen v0.900 peer-review charter;
- the full root DAG and its machine-readable delta;
- a typed rule registry;
- an anti-drift contract and validator;
- a proof kernel, a separately implemented Checker, tests, manifests, and checksums.

## Mandatory rule for every later version

> Every release must include the **entire DAG**, not merely a reference to an older ZIP. It must also include a rule registry and a delta showing every added, removed, renamed, or retyped node. Any silent drift is an obstruction.

## Scope

v0.901 proves only formal conditional results about retained ledgers, kernels, quotient refinement, additive-readout conditions, and observational stationarity. It imports and activates no contingent chemistry equation.
